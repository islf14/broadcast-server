# Chat Server Web App

This web application creates a chat room where anyone can join and send messages. It is not possible for two users to have the same name. When the last user logs out, the chat closes and users are reset.

## Requirements

- Node.js must be installed.

## How to run

Perform the following steps:

- To install dependencies, run in console:

```bash
npm install
```

- To build the project, run in console:

```bash

npm run build
```

- To start the project, run in console:

```bash
npm run start
```

- To use URL Shortening, open in your browser. (port assigned or port 3000)

```bash
http://localhost:3000/
```
